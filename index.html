Project cvslr

Welcome to my first foray into R development! csvlr (Pronounced "Siv-Lur") aims at making the data pre-processing step less painful when working with multiple CSV/text files. This package is made to be used in conjunction with other data wrangling packages like dplyr and tidyr. Currently the package contains 4 functions, but I hope to expand the offerings with each package update.
